AWS 雲端架構與成本計算指南

歡迎使用 AWS Cost & Architecture Lab！這是一個互動式的雲端基礎設施學習與規劃平台。

💡 操作提示

  1. 新增服務：從左側目錄中點擊任何 AWS 服務（如 EC2, RDS），即可將其新增至中央的 VPC 虛擬網路畫布中。
  2. 調整配置：在中央畫布點選該服務卡片，右側會彈出詳細的規格滑桿與選單（如實例規格、儲存大小、流量規模等）。
  3. 獲取基礎設施即代碼 (IaC)：隨着您拉動滑桿變更配置，右側的 Terraform 與 AWS CLI 的部署代碼會同步更新，您可以直接一鍵複製！
  4. 實時成本分析：底部面板將隨時反應您的架構設計變更，通過圓餅圖拆解各類服務的預估費用，並提供對應的 FinOps 成本優化建議。

💰 費率模型概要 (模擬報價)

  • EC2 虛擬主機：基於實例等級計費，`t3.micro` ($7.30/月) 至 `c6g.2xlarge` ($205.86/月)。可開啟預留實例 (RI) 打 6 折。
  • Lambda 函數：按請求量 ($0.20/百萬次) 及記憶體運行時間計費。
  • RDS 資料庫：依主機規格計費，預設開啟 Multi-AZ 高可用部署將雙倍計費。
  • DynamoDB：按寫入/讀取容量單位 (WCU/RCU) 以及儲存量 ($0.25/GB) 計費。
  • S3 儲存：標準儲存單價 $0.023/GB，讀寫 API 按次計費。
